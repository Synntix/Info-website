Semestre 3 - M3302 : Semaine C

Snippets Atom

Ces snippets permet d'importer les librairies principales du C juste en commençant à taper son nom

Pour importer ces snippets : Allez dans Edit > Snippets... et coller le paragraphe suivant :

".source.c":
  "#include <stdio.h>":
    prefix: "stdio"
    body: "#include <stdio.h>"
  "#include <stdlib.h>":
    prefix: "stdlib"
    body: "#include <stdlib.h>"
  "#include <string.h>":
    prefix: "string"
    body: "#include <string.h>"
  "#include <math.h>":
    prefix: "math"
    body: "#include <math.h>"
