Sélecteurs CSS

balise1, balise2
Tous les éléments balise1 et balise2 de la page
balise1 balise2
Les éléments balise2 contenus dans balise1
balise1 > balise2
Les éléments balise2 qui ont pour parent un élément balise1
balise1 + balise2
Les éléments balise2 qui suivent immédiatement des éléments balise1
balise1 ~ balise2
Les éléments balise2 qui précèdent des éléments balise1
#identifiant
L’élément dont l’id est "identifiant"
.nomclass
Tous les éléments qui possèdent la classe "nomclass"
balise.nomclass
Tous les éléments "balise" qui possèdent la classe "nomclass”
balise:first-child
Le premier enfant de la balise "balise"
balise:last-child
Le dernier enfant de la balise "balise"
balise:nth-child(n)
Le nième enfant de la balise "balise"
balise:nth-of-type(n)
La nième balise "balise" de son parent
balise:nth-last-of-type(n)
Les nièmes dernières balises "balise" de leur parents

Positionnement : Flex

Propriétés du conteneur

display: flex;
Tous les enfants du conteneur deviennent des flex items
flex-direction: (row|row-reverse|column|column-reverse)
Définit la direction de l’axe principal du conteneur flex
justify-content: (center|flex-start|flex-end|space-around|space-between)
Contrôle l'alignement de tous les éléments-flex sur l'axe principal
align-items: (center|flex-start|flex-end|stretch)
Contrôle l'alignement de tous les éléments-flex sur l'axe secondaire
flex-wrap: (nowrap|wrap|wrap-reverse)
Décide si les flex-items peuvent aller sur une deuxième ligne si la fenêtre est trop petite

Propriétés des flex-items

align-self: (stretch|flex-start|flex-end|center)
Contrôle l'alignement d'un élément-flex particulier le long de l'axe secondaire
flex: 1;
L'élément prend l'espace restant dans son conteneur
