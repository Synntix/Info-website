Les opérateurs

Projection

Principe résumé :
Permet de ne garder que les attributs (colonnes) choisis
Algébrique :
R ← RA[X]
SQL : 
SELECT X
FROM RA;

Séléction

Principe résumé :
Permet de ne garder que les n-uplets (lignes) remplissant une ou plusieurs condition(s)
Algébrique :
R ← RA / Condition
SQL : 
SELECT …
FROM RA
WHERE Condition;
Illustration :

Jointure

Principe résumé :
Permet de concaténer un attribut (colonne) de deux relation (table)
Algébrique :
R ← RA * (VA = VB) RB
SQL : 
SELECT …
FROM RA, RB
WHERE RA.VA = RB.VB;

Ou

SELECT …
FROM RA
WHERE VA IN(SELECT VB
            FROM RB);
